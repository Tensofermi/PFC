========matlab程序==========
三模主程序：three_mode_V0.m——————其中的路径在不同操作系统下需要修改！！！
绘制结构因子：S_q.m
绘制莫尔条纹：draw.m
辅助绘制莫尔条纹动态图：test.m

========可调参数==========
N,dx,dt,nsteps——计算精度参数
den0,r,lamda——热力学以及初始参数
km1,km2,km3,b1,b2,b3——几何参数
a——立方项，通常为0

=========输出结果=========
1.密度演化过程图以及值
2.能量曲线值
3.衍射图以及结构因子值
4.薄膜、基底、莫尔条纹图
